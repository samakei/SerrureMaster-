2. Run installation commands
3. Verify with `npm test -- --run`

### Write a New Feature

1. Create branch from [CONTRIBUTING.md](CONTRIBUTING.md)
2. Follow [CODE_QUALITY.md](CODE_QUALITY.md)
3. Write tests from [TESTING_GUIDE.md](TESTING_GUIDE.md)
4. Run checks from [SCRIPTS_GUIDE.md](SCRIPTS_GUIDE.md)
5. Submit PR with checklist from [CONTRIBUTING.md](CONTRIBUTING.md)

### Fix a Bug

1. Create issue with details
2. Create branch from [CONTRIBUTING.md](CONTRIBUTING.md)
3. Write test case (reproduce bug)
4. Fix the code
5. Verify test passes: `npm test -- --run`
6. Follow [CONTRIBUTING.md](CONTRIBUTING.md) for PR

1. Review [.github/workflows/tests.yml](.github/workflows/tests.yml)
2. Add environment secrets to GitHub
3. Push test commit
4. Monitor workflow in GitHub Actions
5. Verify status checks pass
